TROUBLESHOOTING UPDATES:
═══════════════════════════════════════════════════════════════════

Edge-TTS not working?
  → Make sure you have internet connection
  → Run: pip install --upgrade edge-tts
  → Check: python generate_voice.py

OpenRouter API error?
  → Check API key in .env file
  → Verify key starts with: sk-or-v1-
  → Test at: https://openrouter.ai/

Different voice not working?
  → Check spelling: en-US-AriaNeural (case sensitive)
  → Test: python generate_voice.py "Hello" "test.mp3"
