📋 MODULE DESCRIPTIONS
──────────────────────

generate_script.py:
  - Uses OpenAI GPT-3.5/4 API
  - Creates engaging short-form scripts
  - Customizable by topic and style
  - Output: 100-500 character scripts
  - Time: ~1-2 seconds per script

generate_voice.py:
  - Uses ElevenLabs API
  - Converts scripts to speech
  - Multiple voice options
  - Supports stability control
  - Output: MP3 audio file
  - Time: ~3-5 seconds per audio

generate_images.py:
  - Uses Pexels API
  - Searches stock images
  - Auto-downloads 10 images
  - Quality optimized
  - Output: JPG images
  - Time: ~5-10 seconds per topic

generate_captions.py:
  - Generates SRT subtitles
  - Time-synced with video
  - Auto sentence splitting
  - Proper formatting
  - Output: SRT file
  - Time: ~1 second

create_video.py:
  - Uses MoviePy library
  - Composes images + audio + captions
  - Optional background music
  - Vertical format (1080x1920)
  - Output: MP4 file
  - Time: ~30-60 seconds per video

🔑 API REQUIREMENTS
───────────────────

OpenAI GPT:
  - Sign up: https://platform.openai.com/
  - Get API key: https://platform.openai.com/api-keys/
  - Env variable: OPENAI_API_KEY
  - Cost: Pay per token (~$0.01-0.05 per script)
  - Free tier: $5 credit

ElevenLabs:
  - Sign up: https://elevenlabs.io/
  - Get API key: Profile → API Key
  - Env variable: ELEVENLABS_API_KEY
  - Cost: Free tier (limited), $5/mo (premium)
  - Usage: ~1-2k characters per short video

Pexels:
  - Sign up: https://www.pexels.com/api/
  - Get API key: Create API key
  - Env variable: PEXELS_API_KEY
  - Cost: COMPLETELY FREE
  - No rate limiting issues

🚀 EXECUTION FLOW
──────────────────

2. Run: python main.py
3. For each topic:
   a. Load topic from CSV
   b. Generate script (GPT)
   c. Synthesize voice (ElevenLabs)
   d. Download images (Pexels)
   e. Generate captions (SRT)
   f. Compose video (MoviePy)
   g. Save to output/
