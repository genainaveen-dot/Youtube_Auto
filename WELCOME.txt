🎯 STEP-BY-STEP SETUP
──────────────────────────────────────────────────────────────────

   STEP 1 - Create directories
   ──────────────────────────
   Run: python setup.py

   This creates:
   • assets/ (backgrounds, music)
   • audio/ (generated audio)
   • captions/ (SRT files)
   • images/ (downloaded images)
   • output/ (final videos) ⭐
   • scripts/ (utilities)
   • topics/ (topics.csv)


   STEP 2 - Install dependencies
   ──────────────────────────
   Run: pip install -r requirements.txt

   Takes 2-5 minutes depending on your internet


   STEP 3 - Configure API keys
   ──────────────────────────
   Edit: .env file

   Add your API keys:
   OPENAI_API_KEY=sk-...
   ELEVENLABS_API_KEY=...
   PEXELS_API_KEY=...


   STEP 4 - Add video topics
   ──────────────────────────
   Edit: topics/topics.csv

   Format:
   topic_id,topic_title,category,difficulty
   1,AI Future,Tech,Medium
   2,Cooking Tips,Food,Easy


   STEP 5 - Generate videos
   ──────────────────────────
   Run: python main.py

   This processes each topic and creates MP4 videos
   Videos saved to: output/ directory

📞 TROUBLESHOOTING
──────────────────────────────────────────────────────────────────
   Module not found?
   → Run: pip install -r requirements.txt

   API errors?
   → Check .env file has correct keys

   Directory errors?
   → Run: python setup.py

   FFmpeg not found?
   → Install FFmpeg for your OS

   No videos?
   → Check topics/topics.csv has data
